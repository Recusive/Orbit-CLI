# codex-rs/skills/src/assets/samples/openai-docs/references/

Reference documentation files for the openai-docs skill.

## What this folder does

Contains markdown reference documents that the agent can consult when answering questions about OpenAI models and APIs.

## Key files

- `gpt-5p4-prompting-guide.md` -- prompting guide for GPT models.
- `upgrading-to-gpt-5p4.md` -- migration guide for upgrading between model versions.
- `latest-model.md` -- information about the latest available model.
