# codex-rs/skills/src/assets/samples/skill-creator/scripts/

Executable scripts for the skill-creator skill.

## What this folder does

Python scripts that automate skill creation and validation.

## Key files

- `init_skill.py` -- initializes a new skill directory structure.
- `quick_validate.py` -- validates a skill package for correctness.
- `generate_openai_yaml.py` -- generates the agent YAML configuration file.
