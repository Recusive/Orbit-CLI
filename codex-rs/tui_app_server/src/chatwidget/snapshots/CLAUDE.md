# codex-rs/tui_app_server/src/chatwidget/snapshots/

Insta snapshot files for chat widget tests.

## What this folder does

Contains `.snap` files generated by the `insta` snapshot testing crate. Each file captures the expected rendered state of the `ChatWidget` (session headers, history cells, streaming output, interrupt prompts, etc.) under specific test conditions.

## What it plugs into

- **../tests.rs**: Test functions in the chat widget test module use `insta::assert_snapshot!()` to compare rendered output against these golden files.
- **insta** (dev-dependency): Manages snapshot creation, diffing, and review.

## Key files

All files follow the naming convention `codex_tui_app_server__chatwidget__tests__<test_name>.snap`.
