# codex-rs/tui_app_server/src/bottom_pane/request_user_input/snapshots/

Insta snapshot files for request-user-input overlay tests.

## What this folder does

Contains `.snap` files generated by the `insta` snapshot testing crate. Each file captures the expected rendered output of the `RequestUserInputOverlay` in a specific test scenario (freeform input, option lists, scrolling, multi-question flows, footer wrapping, etc.).

## What it plugs into

- **../mod.rs** and **../render.rs**: Test functions in these modules use `insta::assert_snapshot!()` to compare rendered output against these golden files.
- **insta** (dev-dependency): Manages snapshot creation, diffing, and review (`cargo insta review`).

## Key files

All files follow the naming convention `codex_tui_app_server__bottom_pane__request_user_input__tests__<test_name>.snap`. Each captures the terminal buffer state for one test case.
