# codex-rs/tui_app_server/src/snapshots/

Insta snapshot files for top-level library module tests.

## What this folder does

Contains `.snap` files generated by the `insta` snapshot testing crate. These capture the expected rendered output of tests defined directly in `src/lib.rs` or other top-level source files (markdown rendering, history cell formatting, diff rendering, etc.).

## What it plugs into

- **../lib.rs** and other top-level `src/*.rs` files: Test functions use `insta::assert_snapshot!()` to compare output against these golden files.
- **insta** (dev-dependency): Manages snapshot creation, diffing, and review.

## Key files

All files follow the naming convention `codex_tui_app_server__<module>__tests__<test_name>.snap`.
