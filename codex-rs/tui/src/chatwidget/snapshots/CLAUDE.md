# codex-rs/tui/src/chatwidget/snapshots/

Insta test snapshots for the ChatWidget test suite.

## What this folder does

Contains `.snap` files generated by the `insta` crate during test runs. These snapshots capture the expected rendered output of `ChatWidget` under various test scenarios including approval modals (exec and patch), streaming output, history rendering, apply-patch manual flows, and more.

## What it plugs into

- **../tests.rs**: Tests in the chatwidget module use `insta::assert_snapshot!()` to verify rendered output against these snapshot files.
- Snapshot files follow the naming pattern `codex_tui__chatwidget__tests__*.snap`.
