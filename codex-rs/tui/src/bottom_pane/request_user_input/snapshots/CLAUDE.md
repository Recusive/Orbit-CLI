# codex-rs/tui/src/bottom_pane/request_user_input/snapshots/

Insta test snapshots for the request_user_input overlay.

## What this folder does

Contains `.snap` files generated by the `insta` crate during test runs. These snapshots capture the expected rendered output of the `RequestUserInputOverlay` widget under various test scenarios, including freeform input, multi-question flows, long option text wrapping, hidden options footers, and footer wrapping.

## What it plugs into

- **../mod.rs**: Tests in the request_user_input module use `insta::assert_snapshot!()` to verify rendered output against these snapshot files.
- Snapshot files are named with the pattern `codex_tui__bottom_pane__request_user_input__tests__*.snap`.
