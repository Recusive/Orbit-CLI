# app-server-protocol/schema/typescript/serde_json

## Purpose

Auto-generated TypeScript type definitions for `serde_json` primitive types used by the app-server protocol. Provides the TypeScript equivalent of Rust's `serde_json::Value`.

## Key Files

| File | Role |
|------|------|
| `JsonValue.ts` | TypeScript type for `serde_json::Value`, mapping to a recursive union of `number | string | boolean | Array<JsonValue> | { [key in string]?: JsonValue } | null`. Used wherever the protocol carries arbitrary JSON payloads. |

## Regeneration

Generated by `cargo run --bin write-schema-fixtures` or `just write-app-server-schema`. Do not edit manually.

## What It Plugs Into

- Imported by other generated TypeScript types in the `typescript/` and `typescript/v2/` directories when they reference arbitrary JSON values.
- Part of the overall TypeScript schema fixture set validated by `tests/schema_fixtures.rs`.

## Exports To

- Other generated TypeScript protocol type files that need a `JsonValue` type.
